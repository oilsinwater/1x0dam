# Tech Stack

## Studio (Sanity CMS)
- Sanity.io Content Management System
- React 17
- TypeScript (implied from dependencies)
- @sanity/base, @sanity/core, @sanity/default-layout, @sanity/desk-tool
- ESLint with @sanity/eslint-config-studio

## Web Frontend (Next.js)
- Next.js 12
- React 17
- React DOM
- @sanity/block-content-to-react for content rendering
- @sanity/client and @sanity/image-url for Sanity integration
- Storybook for component development
- ESLint with eslint-config-next
